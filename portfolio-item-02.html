Title : Shakespear-GPT

Short Subtitle

A neural network that talks like ChatGPT if shakespear taught it english

A 6 layered 256 node neural network that imitates shakespearean writings using token-wise context awareness via self attention including the word frequency and way of usage.

Analyzed the output potential of the model to be 500 tokens, roughly around 3900 words with a base tensor of dimensions 8 x 64 x 256. Extended the model's capability to be integrated with an auto-encoder to create a fully functional Victorian-era ChatGPT.
