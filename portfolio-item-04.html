Title : Disease Prediction using speech

A ML model that predicts diseases from speech

The project was made in collaboration with some of my batchmates for the Intel One-API hackathon.

A random forest model made with Intel-One API as part of their hackathon to input patient symptoms as speech or video and determine illness as model output

tThe model has a 90% accuracy rate and a 3% false positive rate with the current API stats. We also improved the model to take in variable number of inputs and give out accurate multiple potential illnesses, and expanded the dataset to reduce false alerts
