Title : Movie Recommender System

Using Streamlit for website & Cyclic for deployment

Movie Recommender System

The Movie Recommender System is a web app hosted on cyclic. It is built on streamlit and uses some basic UI for the website.

The algorithm that is being used is 'Bag of Words' and the working principle of the project is to use tags for determining the closest movie to a given movie. This is one by creating 5000 dimensional matrix which has frequency of keywords stored withiin. The keywords are determined by director, actors, genre and the synopsis. These 4 arrays are then combined, to form the tags column, and then we can use the created matrix to get every movie as a vector. Then, we just find the closest vector. This in essence is called the Bag of words algorithm.
